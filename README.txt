Getting Started
---------------
1. Open a terminal.
2. Navigate to the root directory of this repository.
3. To initiate the setup or execution process, run the following command:

   sh run.sh

This will execute the `run.sh` script located in the root directory, which is 
designed to compile the code and run it.

If you encounter any issues, ensure that you have the necessary permissions to 
execute the script. You can grant execute permissions using the following command:

   chmod +x run.sh
